Plotter

Vertical plotter is a cnc machine that you can use to transfer digital drawings onto any Vertical surface (ex. a wall, a window). There are few important steps to do: Installing the controller software, Building the Vertical Plotter, Wiring up the components to the Arduino board, Setting up the controller Software, and the last one is loading a Vector Drawing.
